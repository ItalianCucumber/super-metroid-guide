Spring Ball

DESCRIPTION

The Spring Ball is an upgrade to the Morphing Ball. This upgrade is able to let Samus jump while in ball form. Once this
upgrade is attained, this upgrade come very useful throughout the rest of the game. This upgrade is the final Morphing
Ball upgrade attainable in the game.

LOCATION

Maridia

Abilities
