X-Ray Scope

DESCRIPTION

The X-Ray Scope is an upgrade to the Power Suit. This upgrade is able to give Samus sight of hidden blocks such as those that
are breakable or those that you can go through. When you get the X-Ray Scope, you can use it at any time except when mid-air.

LOCATION

Brinstar

X-Ray Scope

Sprite of X-Ray Scope upgrade

Abilities

Allows sight of hidden blocks
