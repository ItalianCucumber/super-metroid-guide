Beams

DESCRIPTION

Beams are an upgraded version of the Power Beam. These beams are able to shoot beams that have different abilities,
such as freezing enemies, hanging from certain blocks and more. Different beam upgrades can be found all across the map,
these beams can be equipped and dequipped as needed. Many different beams can be foundin the game.
These beams range from being able to charge for more power to even shooting through solid blocks
