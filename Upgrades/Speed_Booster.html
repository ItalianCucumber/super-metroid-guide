Speed Booster

DESCRIPTION

The Speed Booster is an upgrade to the Power Suit. This upgrade allows Samus to run extremely quickly, when
sprinting, she kills any enemies in her path and can break certain blocks in her way. Once you find this upgrade,
it is then available to Samus as you go though the rest of the game. Samus only sprints quickly after running for
a certain amount of time, thus limiting where you are able to use this power-up.

LOCATION

Norfair

Speed Booster
